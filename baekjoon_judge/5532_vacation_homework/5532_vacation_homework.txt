5532 방학 숙제


수학 숙제를 하는데 걸리는 날 = (수학 총 페이지수 / 하루에 풀 수 있는 페이지 수) 올림
국어도 같죠?
결국 숙제를 하는데 걸리는 날은 (A / C) 올림 과 (B / D) 올림을 비교해서 높은 쪽이 숙제하는데 걸리는 총 일수가 되겠네요
그걸 L일에서 빼주면 됩니다.
result = L - Max(Ceil(A / C), Ceil(B / D))

질문) 이 문제엔 함정이 하나 있습니다. 뭘까요?

정답) result는 0 이상이어야 한다.

그러므로 마지막에 하나 더 걸어주시면 됩니다.
result = Max(L - Max(Ceil(A / C), Ceil(B / D)), 0)

ps. 그치만 풀어보니 result가 0 이하인 입력이 없다는게...ㅠㅠ
